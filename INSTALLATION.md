# Flutter Installation Log

## Project Goal
Create a Flutter app that works on iPhone

## Installation Steps

### Step 1: Install Flutter SDK
1. Download Flutter SDK from: https://docs.flutter.dev/get-started/install/windows
2. Extract to `C:\flutter`
3. Add `C:\flutter\bin` to PATH environment variable
4. Restart terminal/command prompt

### Step 2: Verify Installation
Run these commands to check installation:
```bash
flutter --version
flutter doctor
```

### Step 3: Install Additional Tools
Flutter doctor will show what else needs to be installed:
- Android Studio (for Android development)
- VS Code with Flutter extension (recommended IDE)
- Chrome (for web development)

## Notes
- For iPhone deployment, will need Xcode on Mac later
- Can develop and test on web browser during development
- Android emulator works on Windows for testing
